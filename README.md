# Evently
**Evently** is a full-stack event planning web application where users can register and log in to create, manage, and RSVP to community events.

## Tech Used

- **Template Engine**: EJS
- **Frontend**: CSS, Tailwind, JavaScript  
- **Backend**: Node.js, Express  
- **Database**: MongoDB

## Installation and Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/AdarshKumar9002/Evently.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Evently
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```
5. Open your browser and visit:
   ```
   http://localhost:<PORT>
   ```
